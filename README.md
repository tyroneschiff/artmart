# Draw Up

![App Coverage](https://img.shields.io/badge/app_coverage-100%25-brightgreen)
![Backend Coverage](https://img.shields.io/badge/backend_coverage-0%25-red)

Parents photograph their children's artwork, AI transforms it into gallery-quality art, and it publishes to a personal store.

## Tech Stack
- **Mobile:** Expo (React Native)
- **Backend:** Supabase
- **AI:** Gemini 2.5 Flash + fal.ai Flux
- **Payments:** Stripe
- **Fulfillment:** Printful

## Autonomous Team
This project is maintained by an autonomous AI engineering team using Gemini CLI.
The team performs a 360-degree audit and implementation cycle every 15 minutes.

## Testing & Coverage
We aim for 100% test coverage. Every new feature must include unit or integration tests.
Tests are run automatically on every push via GitHub Actions.
